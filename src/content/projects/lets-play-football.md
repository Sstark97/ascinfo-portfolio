Aplicación web en la que se pueden ver los jugadores del PSG, con vista detallada de cada uno de ellos. Esta aplicación
se ha desarrollado para una entrevista técnica de React.

## Tecnologías
- React
- Vite
- TypeScript
- TailwindCSS
- React Router
- ESLint
- Prettier

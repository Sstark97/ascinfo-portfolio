# Portfolio API
Guarda tu portfolio en un único sitio para poder utilizarlo donde necesites, utiliza la web para añadir todos tus datos 
y llama a la API desde tus proyectos para utilizar los datos de tu PortFolio.

## Tecnologías
- Python
- Flask
- SQLAlchemy
- Postgres
- Bootstrap
